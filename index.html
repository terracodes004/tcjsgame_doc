Dino Game Documentation (TCJSgame V2)

Creating the Dino Game with TCJSgame V2

Version: 2.0 | Date: March 12, 2025

Description: This Dino Game is built using tcjsgame-v2.js, a lightweight JavaScript game engine for creating 2D games with HTML5 Canvas. The engine provides classes for display management, components, physics, sprites, and utilities that make game development easier.

Game Overview

The Dino game challenges the player to jump over obstacles and survive as long as possible. The score increases over time, and a high score is stored locally using localStorage.

Controls

  • Spacebar – Jump
  • Jump Button – On-screen jump control for mobile/touch users
  • Reset Highscore – Clears stored high score

Main Components

  • Display – Initializes and updates the canvas rendering loop.
  • Component – Used for the Dino, obstacles, score, and text labels.
  • Physics – Gravity and collision detection with crashWith().

Game Logic

  1. Initialization – The Dino and obstacle components are added to the display.
  2. Movement – The obstacle moves left continuously; Dino jumps when triggered.
  3. Collision – If Dino collides with an obstacle, the game resets.
  4. Scoring – Score increases every second; highest score is saved in local storage.

Engine Features Used

  • Canvas-based rendering loop
  • Gravity simulation
  • Sprite and image rendering
  • Collision detection
