Gravity

Gravity is a built-in feature of TCJSgame v3 that simulates the effect of objects being pulled downward, just like in the real world. By assigning a gravity value to a component, you can make it fall until it hits another object, such as the ground or a platform. This is especially useful for platformer games, jumping mechanics, or physics-based puzzles.

Each component has a gravity property. Setting this to a number adds that value to its vertical speed (speedY) each frame. A higher value means faster falling. The gravitySpeed property tracks the total pull of gravity applied so far. To make objects stop when they hit the floor, you usually check for collisions with the bottom of the canvas or with another component.

Syntax

component.gravity = value; // e.g. 0.1

Example

const player = new Component(30, 30, "blue", 50, 50, "rect");
player.gravity = 0.2;
display.add(player);

function update() {
  // The engine applies gravity automatically
  if (player.y > display.canvas.height - player.height) {
    player.y = display.canvas.height - player.height;
    player.speedY = 0;
  }
}

Notes

  • Gravity accumulates in gravitySpeed.
  • Set speedY = 0 when hitting the ground to stop falling.
  • Combine with jump mechanics by setting speedY to a negative value.
