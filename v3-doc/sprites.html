Sprites

Sprites are images that represent characters, objects, or effects in your game. In TCJSgame v3, a Component can be created with the type "image" or "sprite". Sprites allow you to animate characters by cycling through different frames of a sprite sheet. This technique is efficient and commonly used in 2D games.

A sprite sheet is an image that contains multiple frames arranged in a grid. By setting the frameX, frameY, frameWidth, and frameHeight properties, TCJSgame will automatically draw the correct frame. Changing these values over time creates the illusion of animation.

Syntax

const sprite = new Component(width, height, "sprite.png", x, y, "sprite");
sprite.frameWidth = 32;
sprite.frameHeight = 32;

Example

const player = new Component(32, 32, "hero.png", 100, 100, "sprite");
player.frameWidth = 32;
player.frameHeight = 32;
player.frameX = 0;
player.frameY = 0;
display.add(player);

function update() {
  if (display.keys[39]) {
    player.x += 2;
    player.frameX = (player.frameX + 1) % 3; // animate walking
  }
}

Notes

  • Use sprite sheets to save memory and improve performance.
  • Control animation speed by updating frameX every few frames.
  • Sprites can be combined with collisions and gravity just like rectangles.
