Scenes

A Scene in TCJSgame represents a collection of components that are drawn together. Scenes allow you to separate your game into logical sections such as menus, levels, pause screens, and cutscenes. Only one scene is active at a time, and switching between them is extremely simple. This makes it easy to organize complex projects without cluttering your code.

By default, the engine creates Scene 0, which is active when the game starts. When you add a component without specifying a scene index, it is automatically added to Scene 0. If you want to organize your game, you can add components to different scenes and then call display.scene(index) to switch.

Syntax

display.add(component, sceneIndex);
display.scene(sceneIndex);

Example

const menuText = new Component("30px", "Arial", "black", 200, 200, "text");
menuText.text = "Press Space to Start";
display.add(menuText, 0); // menu scene

const player = new Component(30, 30, "blue", 50, 50, "rect");
display.add(player, 1); // game scene

function update() {
  if (display.keys[32]) { // Space key
    display.scene(1); // switch to game
  }
}

Notes

  • Scenes are indexed with numbers starting at 0.
  • Switching scenes hides all other scenes instantly.
  • You can preload assets in one scene and use them later in another.
