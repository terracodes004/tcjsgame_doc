Move Utils

TCJSgame v3 includes several small utility functions to simplify common movement patterns. These are often referred to as Move Utils because they help you move components without having to write repetitive code. For example, instead of manually calculating new positions based on angles, you can call utility methods that handle it for you.

Some utilities include moving an object forward in a given direction, rotating around a point, or following another object. These functions extend what you can do beyond simple x and y adjustments, giving your games more natural motion.

Example: Move Angle

player.moveAngle = 90; // direction in degrees
player.speed = 2;      // movement speed

Example: Follow

function update() {
  enemy.follow(player, 1.5); // move enemy toward player
}

Notes

  • moveAngle works in degrees, where 0 = right, 90 = down.
  • Use speed to control how fast an object moves along its angle.
  • Following is great for enemies, missiles, or homing projectiles.
