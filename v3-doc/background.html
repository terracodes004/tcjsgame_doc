Backgrounds

A good background sets the stage for your game and helps create the right atmosphere. TCJSgame v3 gives you multiple options for backgrounds: solid colors, gradient fills, and even image-based scrolling backgrounds. You can either draw a background as a Component or use the display.background property.

Using image backgrounds allows you to simulate parallax effects where the scenery scrolls at a different speed than the player, giving a sense of depth. Gradients, on the other hand, are perfect for simple skies or abstract designs. Since backgrounds are just components, you can animate them, layer them, or switch them out between scenes.

Syntax

// solid color
display.background = "lightblue";

// gradient background
const gradient = display.context.createLinearGradient(0, 0, 0, display.canvas.height);
gradient.addColorStop(0, "skyblue");
gradient.addColorStop(1, "white");
display.background = gradient;

// image background
const bg = new Component(800, 400, "background.png", 0, 0, "image");
display.add(bg, 0);

Example

const bg = new Component(800, 400, "forest.png", 0, 0, "image");
display.add(bg);

const player = new Component(30, 30, "red", 100, 100, "rect");
display.add(player);

function update() {
  if (display.keys[39]) player.x += 2;
  // move background slower for parallax
  bg.x -= 1;
}

Notes

  • You can use gradients or images for variety.
  • Backgrounds should usually be added to Scene 0.
  • For parallax, move the background at a slower rate than the player.
