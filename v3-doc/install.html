Installation

Since TCJSgame is not available on CDN or NPM, you must include it manually.

  1. Download tcjsgame-v3.js and place it in your project folder.
  2. Include it in your HTML file before your game script.
<script src="tcjsgame-v3.js"></script>
<script src="game.js"></script>
